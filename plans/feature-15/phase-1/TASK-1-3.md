# TASK-1-3: 명세 문서(Spec) 갱신

Gemini CLI Subagent에 대해 새롭게 도입된 `tools: ["*"]` 자동 주입 및 동기화 무시 정책을 관련 기술 명세서에 기록합니다.

## 세부 구현 계획

1. **파일 위치**: `specs/spec.md`
2. **수정 영역**: 
   - **4. 타겟별 변환 사양 (Transformation)** 섹션
     - `Gemini-cli` 타겟의 `에이전트/스킬 변환` 항목 비고 혹은 특이사항 란에 다음 내용을 추가합니다.
       - "Agent 빌드 시 원본 메타데이터에 `tools` 필드가 누락되어 있는 경우 `tools: ["*"]`가 자동으로 주입됩니다."

     - 다음 규칙을 추가합니다.
       - "Gemini CLI의 Agent 동기화 시, 타겟 파일의 `tools` 필드가 `["*"]`이고 원본에 해당 필드가 없었던 경우, `tools` 변경 사항은 무시됩니다."
